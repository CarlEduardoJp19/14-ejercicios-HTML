Glosario Etiquetas HTML
select
Se utiliza para crear menú desplegable que permite al usuario seleccionar una opción.
span
Sirve para diferenciar algo sea importante o no , dentro de un párrafo.
hr
Sirve como línea separadora entre el código o lo que se digite.
p
Lo usamos para añadir algo de contenido y también se le puede meter algún id o una class
table
Usamos el table para crear tablas asimismo que contendrán y modificar el tamaño o contenido.
script
Usamos este src para identificar que haremos una pestaña para el uso de js
form
Utilizamos el form para formularios en los cuales se puede solicitar nombre, correo, contraseña.
link
Usamos este link para poder enlazar nuestra pestaña de html con una pestaña de css para poder darle diseño o color.
ol
Usamos este tipo de lista cuando organizadas.
ul
Usamos este tipo de lista cuando NO es organizada.
